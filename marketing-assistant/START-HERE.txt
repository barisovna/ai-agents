📋 ШАГ 1: НАСТРОЙКА API КЛЮЧА
─────────────────────────────────────────────────────────────────

1. Откройте файл: config.example.php
2. Замените YOUR_DEEPSEEK_API_KEY_HERE на ваш реальный ключ
3. Сохраните как: config.php

Где взять ключ? → https://platform.deepseek.com/api_keys


🧪 ШАГ 2: ПРОВЕРКА РАБОТЫ
─────────────────────────────────────────────────────────────────

Откройте в браузере: https://ваш-домен.ru/test-api.php

Этот файл проверит:
✓ Правильность API ключа
✓ Настройки PHP и CURL
✓ Подключение к DeepSeek API
✓ Покажет тестовый ответ от AI

Если все ✅ зеленое - можно работать!
Если ❌ красное - смотрите инструкции на странице.


🚀 ШАГ 3: ЗАПУСК АССИСТЕНТА
─────────────────────────────────────────────────────────────────

Откройте: https://ваш-домен.ru/marketing-assistant-improved.html

Интерфейс:
• Выберите эксперта (Кривов, Себрант, Захаров)
• Выберите платформы (ВК, Telegram, Дзен, MAX, Яндекс Директ, Авито)
• Выберите режим AI (Chat или Reasoner)
• Опишите ваш проект
• Нажмите "Получить стратегию"


⚙️ РЕЖИМЫ AI
─────────────────────────────────────────────────────────────────

💬 Chat Mode (по умолчанию)
   • Быстрые ответы
   • До 8K токенов (~6000 слов)
   • Подходит для большинства задач

🧠 Reasoner Mode
   • Глубокий анализ
   • До 64K токенов (~48000 слов)
   • Для сложных стратегий


🐛 ОТЛАДКА ПРОБЛЕМ
─────────────────────────────────────────────────────────────────

Если что-то не работает:

1. Проверьте test-api.php - он покажет все проблемы
2. Смотрите логи:
   • api-debug.log - запросы к API
   • api-errors.log - ошибки
3. Консоль браузера (F12) - JavaScript ошибки
4. Читайте README-SETUP.md - подробная инструкция
